h1. ionEngine

h2. Overview

ionEngine is an application framework written in C++ designed primarily for use
in game development and other graphics-heavy applications. The forefront features
are:

# class-based modern OpenGL interface
# highly extensible scene graph supporting culling
# complex event and state management scheme
# robust mathematics library
